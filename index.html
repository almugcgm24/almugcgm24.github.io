protectora de animales (WPF)

En este proyecto colaborativo de grupo, se realizó una aplicación en el entorno Windows destinada a facilitar la gestión de una protectora de animales. Esta aplicación tenía como objetivo posibilitar la gestión continua de información relacionada con los animales, socios, voluntarios y avisos de la protectora.

Para esta aplicación se utilizó la tecnología WPF de Microsoft basada en el lenguaje C# junto con el lenguaje declarativo XAML.
